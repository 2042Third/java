This is a GUI Game of Life simulation program. To start the program follow the guide.

WINDOWS:
  
  -Run a binary excutable version, go to folder "bin\" and run "hw2.exe".
  
  or
  
  -Build and run: 
    Goto scripts folder.
      To build the source code, run script "build.cmd"
      To run the built program, run script "run.cmd"
      To generate javadoc again, run script "javadoccompile.cmd"


MAC or LINUX:
  
  -Run a binary excutable version, go to folder "bin\" and run "hw2.jar".
  
  or
  
  -Build and run: 
    Goto scripts folder.
      To build the source code, run script "build.sh"
      To run the built program, run script "run.sh"
      To generate javadoc again, run script "javadoccompile.sh"


For more information please read the manual.
